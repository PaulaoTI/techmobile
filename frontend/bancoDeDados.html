Banco de dados

Um banco de dados é uma coleção organizada de dados que podem ser armazenados, acessados e manipulados de forma eficiente. Ele permite que informações sejam estruturadas e gerenciadas para diversos fins, como o armazenamento de dados de clientes, inventário, registros financeiros, entre outros.

Principais Conceitos de Bancos de Dados:

  • Tabela: Estrutura que armazena dados em linhas e colunas, semelhante a uma planilha: (ex.: clientes, produtos, pedidos).

  • Registro (linha): Cada linha na tabela representa uma entrada de dados completa para a entidade.

  • Campo (coluna): Cada coluna representa um atributo específico da entidade.

  • Chave Primária: Um campo que identifica exclusivamente cada registro em uma tabela.

  • Chave Estrangeira: Um campo que cria um vínculo entre duas tabelas, estabelecendo relacionamentos.

    • Tipos de Bancos de Dados:

  • Relacional (SQL): Baseado em tabelas, usa a linguagem SQL para manipulação de dados: MySQL, PostgreSQL, Oracle.

  • Não Relacional (NoSQL): Ideal para dados não estruturados, como MongoDB e Cassandra.

  • Banco de Dados em Nuvem: Hospedado em servidores na nuvem, com acesso online: Amazon RDS, Google Cloud Firestore.

  • Banco de Dados Orientado a Objetos: Combina conceitos de orientação a objetos e banco de dados: db4o e ObjectDB.

    • Principais Linguagens:

  • SQL: A linguagem padrão para bancos de dados relacionais, utilizada para criar, ler, atualizar e deletar dados.

  • NoSQL: Como o MongoDB Query Language (MQL), para bancos de dados não relacionais.
